## Deployment

### Option 1 — Drag & Drop
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag all files into the drop zone
3. Live instantly

### Option 2 — GitHub + Netlify (recommended)
1. Push this repo to GitHub
2. Go to [app.netlify.com](https://app.netlify.com) → Add new site → Import from Git
3. Select this repo → Deploy
4. Auto-deploys on every push

### Adding Your Profile Photo
Drop your photo in the root folder and name it `profile.jpg`.
The site picks it up automatically.

### Enabling Contact Form Emails
After deploying on Netlify:
1. Go to your site dashboard → **Forms**
2. Click the `contact` form → **Form notifications**
3. Add `werleedautane@gmail.com` as the notification email
